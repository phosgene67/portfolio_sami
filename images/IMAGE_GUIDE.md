# Portfolio Images Guide

This folder contains all images for the Sami Chowdhury robotics portfolio.

## Required Images

### Profile & Hero Section
- `hero-image.png` - Your profile/robotics workspace image (recommended: 400x400px)
- `about-image.png` - About section image (recommended: 400x500px)

### Project Images
- `rov-project.png` - Underwater ROV project (recommended: 600x400px)
- `uav-project.png` - Autonomous UAV project (recommended: 600x400px)
- `ai-automation-project.png` - AI automation system (recommended: 600x400px)
- `sensor-network-project.png` - IoT sensor network (recommended: 600x400px)

## Image Specifications

### Recommended Formats
- **PNG** - Best for screenshots, diagrams, and transparent backgrounds
- **JPG** - Best for photographs (use 85-90% quality)
- **WebP** - Modern format with better compression (optional)

### Image Dimensions
- **Project Cards**: 600px × 400px (aspect ratio 3:2)
- **Hero Image**: 400px × 400px (square, centered)
- **About Image**: 400px × 500px (portrait, rounded corners)

### File Size Optimization
- Keep file sizes under 300KB for web performance
- Use tools like:
  - TinyPNG (https://tinypng.com/)
  - ImageOptim
  - Squoosh (https://squoosh.app/)

## How to Add Images

1. **Take or create images** of your projects
2. **Resize and optimize** to specifications above
3. **Save as PNG or JPG** with descriptive names
4. **Place in this folder** (`/images/`)
5. **Test responsive design** on mobile devices

## Placeholder Strategy

While you're gathering images, the portfolio will display:
- Gray placeholder boxes
- Text overlays indicating missing images
- All functionality remains intact

Once you add images, they'll automatically display without code changes.

## Tips for Better Project Photos

- **ROV**: Take underwater shot or high-quality photo with camera/lights in frame
- **UAV**: Flying shot, or drone platform from different angles
- **AI Automation**: System architecture diagram or hardware setup photo
- **IoT Sensors**: Sensor nodes, data dashboard, or network visualization
